# Voquill v1.3.0

- Added first-class Linux display server support for both Wayland and X11.
- Improved Linux compatibility for GNOME-based distros including Fedora.
- Refactored Config and Setup UI flows with clearer setting grouping and card-based layout.
- Standardized config/setup action button styling for a more consistent interface.
- Improved hotkey registration reliability, including punctuation combos like `Ctrl + .` and `Ctrl + ,`.
- Added safer hotkey apply behavior with rollback when registration fails.
- Expanded diagnostics and setup tooling in Config, including session log actions and setup entry points.
- Added broader UI interaction tracing (button presses, switch toggles, committed setting changes) to simplify issue debugging.
- Improved cross-platform build reliability and release tooling for Windows and Linux packaging.
