# Voquill v1.3.10

## Highlights
- Reworked the Settings experience with a new high-priority General section and cleaner grouping for behavior vs troubleshooting controls.
- Renamed the Config navigation route to Settings (`#/settings`) for consistent UI terminology.
- Added cross-platform auto-start support (Windows/Linux) with a dedicated Settings toggle.
- Added top-right Experimental badges for in-progress features and moved Turbo Mode into Transcription directly below Local Model.
