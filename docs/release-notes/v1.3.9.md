# Voquill v1.3.9

## Highlights
- Fixed a Fedora GNOME Wayland push-to-talk edge case where release-order quirks could leave recording latched on.
- Added a repeat-activation heartbeat fallback in the Wayland GlobalShortcuts portal path so recording reliably stops and transcribes even when `Deactivated` is delayed/missed.
- Kept normal portal behavior intact (`Activated` starts, `Deactivated` stops) and documented the Fedora-specific rationale and thresholds in the compatibility docs.
